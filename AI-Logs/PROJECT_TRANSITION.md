## 📥 Installation

### Quick Start
```bash
git clone https://github.com/go-while/rocksolid-light.git
cd rocksolid-light/Rocksolid_Light
sudo ./debian-install.sh  # or freebsd-install.sh
```

### For existing installations
Your existing Rocksolid Light installation can be updated to use the community version. See the upgrade documentation in the repository.
